# Suspected Data Exfiltration

## Immediate actions

1. Block export identity and egress route.
2. Preserve database and network audit logs.
3. Determine data scope and legal notification duties.
4. Rotate affected credentials.

## Closure requirements

- Incident record and timeline complete.
- Data-integrity reconciliation PASS.
- Credentials rotated where applicable.
- Corrective actions assigned and tracked.
